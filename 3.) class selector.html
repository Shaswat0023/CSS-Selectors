We see that we can assign a class to a table and can change the appearance of the table, table heading, table rows and table data of a particular table.


Id selector is same as class selector. Instead of assing a class to a particular element we assign a unique id to that particular element by typing id="idname" and an id will be given to that element. To select that id is css instead of using .classname we use #idname and than we can apply the propertie. ID has more specificity than a class.
