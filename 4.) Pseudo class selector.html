A css pseudo class is a keyword added to a selector that specifies a special state of the selected elements. for example, :hover can be used to change a button's color when the user's pointer hovers over it.




Hover Propery

There are many more pseudo class selectors like link, visited and active. Point to be noted is that a:hover MUST come after a:link and a:visited in the CSS definition in order to be effective. a:active MUST come after a:hover in the CSS definition in order to be effective.
