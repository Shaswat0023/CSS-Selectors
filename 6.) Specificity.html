Specificity

  • Because elements can have multiple CSS selectors, there is a hirarchy for the weight given to each type of the selector. Here is the logical order of selectors from least to most weight
  • Type Selectors eg (h1) and pseudo-elememts eg ()::before)
  • Class Selectors eg( .example), attributes selectors eg([type="radio"]), pseudo classes eg (:hover)
  • ID selectors eg(#example)
